Ruby On Rails Webstore

Ruby On Rails Webstore

Project Details

This is a fully production ready webstore built on Ruby on Rails. Users are able to sign up, make payments and write reviews for products sold on the website. It also features a sign up mailer, a fully functional AngularJS order page, and a pop-out sidebar made completely out of CSS!

Gems & Plugins Used

  • Devise (Authentication)
  • CanCanCan (Authorization)
  • will_paginate (Pagination)
  • Raty jQuery plugin
  • RSpec
  • FactoryGirl (generating test data)
  • Guard (automate tasks)
  • Guard RSpec (automate RSpec tests)
  • Spork Rails (faster testing)
  • Guard-Spork
  • Stripe (Payment Processing)
  • AngularJS-Rails (Orders page)
  • Memcached
  • Redis

    • The site also features an Angular JS page displaying existing order info, while allowing new orders to be added, just to test the functionality of the Angular integration. I also used a variety of jQuery tweaks and plugins to improve the sites usability.
