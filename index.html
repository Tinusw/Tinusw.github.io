Skills and Experience

Web Development

Whether you need a new website, or updates to an existing one. if it involves the web, I can help.

I write code that conforms to the W3C's web standards and semantic best practices. I use technologies like HTML5, CSS3, jQuery and AngularJS, to transform static artwork into mobile first, pixel prefect, interactive and fully functional interfaces.

Responsive Design

Smartphones, tablets, even your TV have web access. Your site should be just as adaptable.

How do you keep a brand consistent, while accommodating for various existing and emerging devices? I build using a responsive, mobile first grid systems, built on frameworks such as Bootstrap and Toast to ensure smarter, more future-proof sites.

Source Control

It's very easy to start building without considering consistency, or future development.

I'm familiar with the principles of source control and strive to write clean, maintainable code. I use tools like Git and services like GitHub to ensure consistency in my code, to conform to best practices and to keep code coming any time of the day, Wether alongside a team, or remotely.

Backend Development

While front end engages your customers, the backend drives your application performance.

I know what happens under the hood of your Rails applications. From creating Models & Controllers, through to routing, caching and setting up AJAX requests. I only use the most mature gems to ensure your applications are secure and ready to go live.
